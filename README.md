# geodeterminants

<!-- badges: start -->
<!-- badges: end -->

The goal of geodeterminants is to take addresses and append SDOH(social
determinants of health) data in a tibble format for each specific
address. This data is collected from sources from PhenX for each
variable and is cleaned, analyzed, and calculated, ready for use.

## Installation

You can install the development version of geodeterminants from
[GitHub](https://github.com/wchan05/geodeterminants) with:

``` r
# install.packages("pak")
pak::pak("wchan05/geodeterminants")
```
